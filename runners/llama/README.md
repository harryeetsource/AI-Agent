# llama.cpp runner placement

Place your local llama.cpp binaries here.

Windows example:
- `llama-server.exe`
- `llama-cli.exe`

Linux/macOS example:
- `llama-server`
- `llama-cli`

The helper scripts expect the server binary at:
- Windows: `runners/llama/llama-server.exe`
- Unix: `runners/llama/llama-server`
